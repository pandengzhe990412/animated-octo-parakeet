# 📖 扩展使用说明

## 🎯 如何使用扩展

### 当前状态说明

你现在看到的提示消息：
```
请打开飞书文档页面，然后点击刷新按钮。

支持的域名：
- *.feishu.cn
- *.larksuite.com
```

**这是正常的！**因为你当前可能不在飞书文档页面上。

---

## 📝 完整使用流程

### 第 1 步：打开飞书文档

在浏览器中打开任意飞书文档，例如：
```
https://feishu.cn/doc/xxxxx
```

**或者**新建一个测试文档：
```
https://feishu.cn
```

### 第 2 步：等待页面加载

确保文档完全加载，能看到：
- ✅ 文档标题
- ✅ 正文内容
- ✅ 编辑器界面

### 第 3 步：打开扩展侧边栏

1. 点击 Chrome 右上角的扩展图标
2. 侧边栏会从右侧滑出

### 第 4 步：抓取文档内容

#### 方式 A：手动刷新（推荐）

点击侧边栏底部的 **"刷新"** 按钮

#### 方式 B：自动抓取

部分版本会自动抓取（如果实现的话）

### 第 5 步：查看结果

成功后应该看到：
- 📱 **文档标题** - 显示在顶部状态栏
- 📄 **预览内容** - 手机预览框显示内容
- 🎨 **主题选择** - 可以切换主题
- 📋 **复制按钮** - 可以复制到公众号

## ⚠️ 如果还是失败

### 检查清单：

1. **确认在飞书页面**
   - URL 必须包含 `feishu.cn` 或 `larksuite.com`
   - 不是登录页，不是首页，是具体的文档页

2. **确认页面完全加载**
   - 等待几秒让页面完全加载
   - 可以刷新页面（F5）

3. **确认扩展权限**
   - 打开 `chrome://extensions/`
   - 找到扩展，点击"详细信息"
   - 检查权限：
     - ✅ 存储数据
     - ✅ 活动标签页
     - ✅ 剪贴板写入
     - ✅ 侧边栏

4. **重新加载扩展**
   - 在扩展页面点击"重新加载"按钮
   - 切换回飞书文档页面
   - 点击扩展图标

## 🎨 功能说明

### 1. 主题切换

点击底部的主题下拉菜单：
- **科技蓝** - 蓝色主题，适合技术文章
- **简约白** - 白色主题，通用风格
- **商务黑** - 黑色主题，正式商务

### 2. 编辑/预览切换

点击顶部状态栏的按钮：
- **编辑** - 显示 Markdown 源码，可以手动编辑
- **预览** - 显示手机预览效果

### 3. AI 润色（开发中）

点击"AI 润色"按钮（功能开发中）

### 4. 复制到公众号

点击"复制"按钮：
- 自动应用当前主题样式
- 转换为微信兼容格式
- 可以直接粘贴到公众号编辑器

---

## 💡 使用技巧

### 快速切换

1. 在飞书文档和公众号编辑器之间切换
2. 使用快捷键：
   - `Alt + T` - 打开侧边栏（如果设置了快捷键）
   - `Ctrl + C` - 复制内容
   - `Ctrl + V` - 粘贴到公众号

### 批量处理

如果有多个飞书文档：
1. 打开第一个文档
2. 点击扩展，抓取内容
3. 点击"复制"
4. 粘贴到公众号
5. 关闭侧边栏
6. 重复以上步骤

## 🚨 错误消息说明

### "请打开飞书文档页面"

**原因**：不在飞书页面

**解决**：
1. 打开 `https://feishu.cn/doc/xxxxx`
2. 点击"刷新"按钮

---

### "无法获取文档内容"

**原因**：页面选择器找不到内容

**解决**：
1. 确保在文档编辑页面（不是列表页）
2. 刷新页面（F5）
3. 等待完全加载后，点击"刷新"按钮

---

### "获取失败"

**原因**：权限或跨域问题

**解决**：
1. 检查扩展权限
2. 重新加载扩展
3. 查看控制台错误信息
