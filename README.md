# 油管短剧挖掘机 Pro

一款专业的 YouTube 短视频监控和下载浏览器扩展，帮助你追踪对标账号的最新内容，第一时间发现爆款视频。

## 功能特点

### 1. 对标账号监控
- 添加任意 YouTube 频道到监控列表
- 后台定时检查（可配置 1-120 分钟）
- 自动发现新发布视频
- 智能预警：48小时内的新视频提醒

### 2. 快速下载
- 浏览器右键菜单快速下载
- 支持标准视频和 YouTube Shorts
- 无需本地服务器，使用在线下载服务

### 3. 数据分析
- 查看监控频道的最新视频列表
- 按发布时间、播放量排序
- 一键跳转下载视频

### 4. 独立控制台
- 完整的监控管理界面
- 实时查看最新发现内容
- 浏览器通知提醒

## 安装方法

### Chrome/Edge 浏览器

1. 下载本项目到本地
2. 打开浏览器扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹

## 使用说明

### 快速下载视频

1. 点击浏览器工具栏的扩展图标
2. 粘贴 YouTube 视频链接（支持自动识别当前页面）
3. 点击"在线下载"按钮
4. 在打开的下载页面中选择格式下载

### 监控对标账号

1. 点击扩展图标 → "打开控制台"
2. 进入"对标监控"页面
3. 添加要监控的频道链接
4. 开启监控开关
5. 设置检查间隔（建议 30-60 分钟）

### 查看监控结果

- 在控制台查看最新发现的视频
- 收到通知时点击"查看详情"
- 支持按频道筛选、一键下载

## 技术栈

- Chrome Extension Manifest V3
- Vanilla JavaScript（无依赖）
- YouTube Feeds API（获取频道视频）
- Chrome Storage API（数据持久化）
- Chrome Alarms API（定时任务）
- Chrome Notifications API（系统通知）

## 常见问题

**Q: 为什么需要开发者模式？**
A: 本扩展未上架 Chrome 应用商店，需要手动安装。

**Q: 下载功能如何实现？**
A: 使用在线下载服务，无需安装额外软件。

**Q: 监控多久检查一次？**
A: 可在控制台配置，建议 30-60 分钟，避免频繁请求。

**Q: 会影响浏览器性能吗？**
A: 不会。监控在后台运行，仅定期获取少量数据。
